# Health Risk Prediction

## 📌 Project Description
This project focuses on analyzing lifestyle and health data to identify key factors affecting disease risk and to build a machine learning model for prediction.

## 🎯 Goal
To develop a model that predicts the risk of disease based on user lifestyle and physiological data.

## 🧠 Technologies
- Python (pandas, numpy, scikit-learn)
- Data preprocessing and feature engineering
- Machine Learning (Logistic Regression, Random Forest, Gradient Boosting)
- SMOTE (handling imbalanced data)
- Data visualization (DataLens, matplotlib, seaborn)

## 🔍 Steps
1. Data cleaning and preprocessing  
2. Exploratory Data Analysis (EDA)  
3. Feature engineering  
4. Handling imbalanced data using SMOTE  
5. Model training and comparison  
6. Model evaluation using ROC-AUC and accuracy  

## 📊 Results
The model is able to predict disease risk based on lifestyle and health indicators.

## 📈 Dashboard
An interactive dashboard was created to visualize key insights and model results.
